<Sobre mí/>

Soy un apasionado del aprendizaje constante y la superación personal. Recientemente he finalizado mis estudios como desarrollador backend, lo que representa un paso importante en mi trayectoria profesional. Durante este proceso, he adquirido habilidades sólidas en tecnologías clave del desarrollo web y backend, como la gestión de bases de datos, la creación de aplicaciones eficientes, y la integración de sistemas.

Antes de adentrarme en el mundo del desarrollo, estudié Administración y Finanzas, donde aprendí a gestionar recursos, analizar datos y planificar estrategias. Esta formación me dio una base sólida en organización y análisis crítico.

Una parte significativa de mi vida la dediqué a la cocina, un entorno que me enseñó la importancia del trabajo en equipo, la atención al detalle y la creatividad para resolver desafíos diarios.

Ahora, estoy buscando una oportunidad para aplicar todo lo aprendido en un entorno profesional que me permita seguir creciendo, aprendiendo, y contribuyendo al éxito de un equipo. Estoy convencido de que mi combinación de experiencia en áreas diversas, junto con mi formación técnica, me permite aportar una perspectiva única y valiosa a cualquier proyecto.

360planet

Plataforma integral desarrollada desde cero para la gestión y envío masivo de mensajes de WhatsApp, orientada a clínicas y empresas. Incluye paneles web de administración y cliente avanzados, así como aplicaciones móviles multiplataforma (Kotlin Multiplatform Mobile para Android/iOS), permitiendo gestionar usuarios, compras y facturación.

Integra pagos seguros mediante Stripe y PayPal, generación y descarga automática de facturas en PDF personalizadas, autenticación de usuarios y exportación de datos en CSV. El backend en Node.js + Express usa SQLite y automatiza el envío de mensajes de WhatsApp a través de Twilio API (WABA).

Destaca la sincronización en tiempo real de compras y mensajes entre el panel web y las apps móviles, la gestión de sesiones, estadísticas, así como el envío de recordatorios y notificaciones automáticas por correo electrónico mediante Nodemailer.

Node.js Express SQLite Stripe PayPal Twilio API Kotlin Multiplatform Jetpack Compose SwiftUI Bootstrap Puppeteer Swift Xcode

Donde está mi coche

Aplicación móvil avanzada diseñada para facilitar a los usuarios la localización de su vehículo, guardando la ubicación exacta del aparcamiento y permitiendo añadir notas y fotos. Integra Google Maps para mostrar rutas de regreso, recordatorios inteligentes para evitar olvidos y un historial de ubicaciones.

Destaca por incluir un asistente de ayuda basado en IA, accesible mediante un chat integrado, que resuelve dudas sobre el uso de la app y guía al usuario de forma interactiva. Además, permite la sincronización en la nube con Firebase, ofreciendo respaldo y recuperación de datos entre dispositivos. La gestión local de datos ha sido optimizada migrando de archivos JSON a SQLite para mayor rendimiento y fiabilidad.

Android Kotlin Google Maps API SQLite Firebase IA/Chatbot

LinkTracker

Aplicación desarrollada en JavaFX que permite cargar archivos de texto con URLs de páginas web, procesar múltiples hilos concurrentes y recopilar los enlaces contenidos en esas páginas. La aplicación utiliza un diseño intuitivo con ListViews para mostrar las páginas procesadas y los enlaces encontrados.

Java JavaFX Threads Concurrent Programming

FlightsFX

FlightsFX es una aplicación desarrollada en JavaFX diseñada para gestionar una lista de vuelos de un aeropuerto. Los usuarios pueden cargar vuelos desde un archivo de texto, añadir nuevos vuelos, eliminarlos, y aplicar diferentes filtros como vuelos largos, próximos 5 vuelos, y vuelos a un destino seleccionado.

Java JavaFX FXML Streams File I/O
